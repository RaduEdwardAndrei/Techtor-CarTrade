# Techtor-CarTrade
  Techtor is a mentoring program organized by Codette in partnership with the Lenovo Foundation. For 6 weeks I chose to work on a project in which I will create a car sales website with different functionalities.


---*MIDTERM*---

  Pana in momentul de fata, am reusit sa creez un formular de Login/Sing_Up pe care urmeaza sa il integrez in frameworkul de Laravel.
Am folosit Html,Css si Javascript pentru functionalitatea de Slide-In a formularului si php pentru a prelua informatiile si sa le adaug in baza de date folosind MYSQL plus mesaje de eroare in cazul in care nu este completat formularul in mod corect.
